Modern WordPress theme built with React 19 and Vite.

== Description ==

Enspyred Theme is a modern, React 19-based WordPress theme built with Vite and Styled Components. It provides a fast, component-driven development experience for WordPress theme development.

**Features:**
- React 19 with modern hooks
- Vite 6 for lightning-fast builds
- Styled Components for CSS-in-JS styling
- Hot module replacement during development
- Clean, production-optimized builds

**Technology Stack:**
- React 19
- Vite 6
- Styled Components 6
- WordPress theme integration

== Installation ==

= IMPORTANT: Download the Correct ZIP File =

**DO NOT** download the repository ZIP from the main branch! The repository includes development files (node_modules, source code, etc.) totaling ~129MB and will not work properly when installed in WordPress.

**ALWAYS** download the official release ZIP from GitHub Releases. These are clean, production-ready distributions (~2-3MB) that contain only the necessary files.

= Installation Steps =

1. Download the latest **release ZIP** from GitHub: https://github.com/enspyred/wp-theme-enspyred-theme/releases
2. Log in to your WordPress admin panel
3. Navigate to Appearance > Themes > Add New > Upload Theme
4. Choose the downloaded release ZIP file and click "Install Now"
5. Activate the theme

== Frequently Asked Questions ==

= Can I download the theme from the GitHub repository? =

No! You must download the official release ZIP from GitHub Releases. The repository contains development files that are not suitable for WordPress installation.

= What PHP version is required? =

PHP 8.0 or higher is required.

= What WordPress version is required? =

WordPress 6.0 or higher is recommended.
